# 月次オペレーション分析レポート機能ガイド

**関連 Issue**: #228
**最終更新**: 2026-03-27

---

## 概要

クライアントの仕訳データを基に、Anthropic API（Claude）で月次オペレーション分析レポートを自動生成・保存・閲覧する機能。
業種（宿泊業/飲食業/アクティビティ/その他）に応じて分析観点を自動切り替えする。

---

## 画面遷移フロー

```
クライアント一覧 (/clients)
    │
    ▼
クライアント詳細 (/clients/:code)
    │
    ├── [月次レポート] カード
    │       │
    ▼       ▼
月次レポート一覧 (/monthly_reports?client_code=xxx)    ← サイドバーからも遷移可能
    │
    ├── [レポート生成] ボタン → POST /monthly_reports/generate
    │                              │
    │                              ▼ (成功時)
    ├── [閲覧] リンク ──────→ レポート詳細 (/monthly_reports/:id?client_code=xxx)
    │                              │
    │                              ├── [再生成] ボタン → POST /monthly_reports/generate
    │                              ├── [印刷] ボタン → window.print()
    │                              ├── [削除] ボタン → DELETE → 月次レポート一覧
    │                              └── [← 一覧に戻る] → 月次レポート一覧
    │
    └── [削除] ボタン → DELETE /monthly_reports/:id → 月次レポート一覧
```

## 業種別分析観点

| 業種 | industry値 | 主な分析観点 |
|------|-----------|-------------|
| 宿泊業 | hotel | 稼働率、客単価、清掃コスト比率、季節変動、設備メンテ、光熱費、OTA手数料 |
| 飲食業 | restaurant | 原価率、売上推移、仕入先別分析、食材ロス、人件費率、光熱費、販促効果 |
| アクティビティ | tour | 参加者数推移、天候連動、安全関連費用、車両機材費、季節プログラム、集客チャネル |
| その他 | nil等 | 汎用（収支バランス、固変分析、キャッシュフロー、主要取引先） |

## 利用方法

### Web UI から手動生成

1. クライアント詳細画面の「月次レポート」カードをクリック
2. 対象年月を選択し「レポート生成」をクリック
3. AI がレポートを生成（数十秒）
4. 完成したレポートが詳細画面で表示される

### Rake タスクで一括生成

```bash
# 全activeクライアントの前月分を一括生成
make test  # ではなく以下を実行
docker compose -f compose.development.yaml --env-file .env.development exec platform bash -lc "bin/rails reports:generate_monthly"
```

### Claude Code skill で手動生成

```
/monthly-report
```
